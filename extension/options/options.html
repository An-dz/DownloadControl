    Let's start with a quick initial setup that's required for this extension to work properly.
    You can close this window to skip these 3 steps for now, but some functionality will not be working until you complete it.

    Step 1

    Download Control will determine a folder for each download you start according to the rules you set up
    Due to security restrictions this is only possible inside your browser's default download folder. As a consequence, you need to choose this folder in a way that it contains all folders that you want to save files at.
    e.g. if you want to save files at "C:\Users\You\Downloads\Stuff\" and "C:\Users\You\Documents\", the perfect choice would be "C:\Users\You\".

    To specify this folder, go ahead now and
    1. open settings (Alt+P)
    2. scroll down to see "Download location"-setting in "Downloads"-section
    3. click "Change..."
    4. select the folder you want to use

    Step 2

    Download Control can make your life easy
    It can offer you to add a new rule whenever you choose to save a file at a location different from the one it expects this file to be saved at.
    If you don't need or want this feature, you can proceed with step 3.
    If you do want this feature to work, you need to tell Download Control the folder you just chose in step 1. Enter it in here:




    Note: Download Control will not work properly if this folder is incorrect. Remember to always change it in here if you change it in your browser's settings!

    Step 3

    A universal rule
    Changing your browser's default folder comes along with one disadvantage: your default folder changed (Oh, really?).
    In order to prevent all files that don't get matched by any of your rules from ending up in this folder, you need to tell Download Control the subfolder to save those at.
    e.g. if you want your default folder to be "C:\Users\You\Downloads\" and you set "C:\Users\You\" in step 1 (and 2), enter "Downloads\".



    The initial setup is completed.
    If you want to, you can start creating some rules or have a look at Download Control's settings now.
